# ESP_WiFi_Smart_config

1. เปิดโปรแกรมตัวอย่าง `smart_config`
2. build, flash และ monitor โปรแกรม
3. ดาวน์โหลด application `ESPTouch`
4. เชื่อมมือถือเข้ากับ WiFi access point
5. ทำการ config โดยการป้อน WiFi password
